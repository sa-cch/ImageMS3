Change JDBC to SpringJDBC

JDBC→SpringJDBCの変更について整理

そもそもJDBCとは。

Java Database Connectivityの略。
Javaで書いたプログラムとデータベースを接続してくれるもの。
※データベース製品ごと（MySQL, PostgreSQL, Oracle DataBaseなど）の違いを吸収してくれるので、統一的な操作が可能。べんり！

SpringJDBCに変更したらどんなメリットがある？

①シンプルに記述が楽になる
②例外処理を書き換えなくて良い

メリット①：シンプルに記述が楽になる について

Templateクラスを使うことで記述が楽に。
・JdbcTemplateクラス
・NamedParameterJdbcTemplateクラス
↑Connection, Statement, ResultSetの準備やクローズ処理の記述が不要になる。

メリット②：例外処理を書き換えなくて良い について

データベース製品ごとに異なるデータアクセス例外を、Springで定義した共通のデータアクセス例外に変換してくれるので、統一的な記述が可能。べんり！
